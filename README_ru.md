# Mods Dependencies Scanner
Данное приложение позволяет быстро вывести список модификаций-зависимостей всех модификаций в Minecraft сборке, без запуска самого Minecraft и последующего чтения Crash-log. <br>
Приложение консольное(не имеет GUI, управление происходит через консоль). После запуска приложение потребует ввести путь до директории(папки) с модификациями, список зависимостей которых необходимо вывести. А после выведет весь список необходимых, модификации требующие их, а так же требуемую их версию.

На данный момент приложение поддерживает лишь анализ модификаций созданных под загрузчик Forge, для Minecraft версии выше 1.14. Но в будущих обновлениях я реализую поддержку всех типов Minecraft модификаций, под все типы загрузчиков.

## Загрузка и запуск
Для запуска приложения загрузите нужную вам его версию на странице релизов, либо просто скачайте последнюю его версию по кнопке ниже:<br>
<a href="https://github.com/SlavaEnergy/Mods-Dependencies-Scanner/releases/download/file/Mods.Dependencies.Scanner-1.0.jar">
  <img src="https://img.shields.io/badge/Download-JAR-blue?style=for-the-badge">
</a>
<br>
После переместите скачанный **.jar** файл любую директорию, из которой вы будете его запускать(или оставьте в той же).<br>
Запустите любую поддерживающую консоль, например CMD.exe(если ваша операционная система - Windows).<br>
Перейдите в директорию, в которую вы поместили скачанный файл (команда зависит от выбранной вами консоли).<br>
А после выполните в ней:<br>
`java -jar "Mods Dependencies Scanner-1.0.jar"` , изменив версию в команде на версию загруженного файла.<br>
Это запустит программу, дальше следуйте требованиям самой программы. Как только программа завершит своё действие, для повторного запуска необходимо ввести команду заново.<br>
**Внимание:** для запуска программы требуется установленный в системе JRE Java любой поддерживаемой версии, но я рекомендую Java 21.

## Компиляция
Если вы хотите заново скомпилировать jar файл из предоставленных здесь исходных файлов, вы можете проследовать дальнейшей инструкции, либо при наличии навыков выполнить это самостоятельно:<br>
Для компиляции стоит выгрузить проект, как zip архив, распаковать его, а после выполнять дальнейшие действия с ним, либо клонировать этот репозиторий и работать уже с ним.<br>
Для компиляции, просто выполните из директории проекта следующую команду:<br>
`gradle wrapper` ,<br>
а затем - эту:<br>
`gradle build`.<br>
**Внимание:** для компиляции данной программы вам потребуется установленный в системе Gradle, а так же JDK Java 21 (либо любой другой поддерживаемой версии, но в таком случае вам прийдётся отредактировать файл build.gradle).<br>
